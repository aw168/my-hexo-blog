## 第一步：开始工作前，先“听”一次

- 在您准备修改任何东西之前，先运行 `pull` 命令

```bash
git pull origin main
```

- **本地预览**

```bash
hexo cl; hexo s
```

- ​**提交所有修改**​：

```git
git add .
git commit -m "修改的大致内容"
```

- **修改后后推送**

```bash
git push origin main
```
